# Личные финансы — Finance Tracker

**Версия:** 1.0.2  
**Язык:** Python 3.12  
**Хранение данных:** CSV (встроенный модуль `csv`)  
**Тестирование:** pytest + pytest-cov

---

## Описание

Консольное приложение для учёта личных доходов и расходов.
Позволяет контролировать бюджет, анализировать траты по категориям и отслеживать
норму сбережений. Реализует полный цикл разработки: от ТЗ и TDD до приёмочного
тестирования и имитации поддержки.

## Инструкция по запуску

```bash
# 1. Клонировать репозиторий
git clone https://github.com/your-username/finance-tracker.git
cd finance-tracker

# 2. Создать виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate  # Linux/Mac

# 3. Установить зависимости
pip install pytest pytest-cov

# 4. Запустить приложение
python menu.py

# 5. Запустить тесты
pytest tests/ -v

# 6. Проверить покрытие кода
pytest tests/ --cov=. --cov-report=term-missing
```

## Расчёт нормы сбережений

```
savings_rate = (balance / income) × 100%
balance = income − expense
```

Считается нормой экономить 20% и более (правило «50/30/20»).

## FAQ

**Q: Как сбросить все данные?**  
Удалите файл `data/finance_log.csv` или выполните:
```python
python -c "import data_service; data_service.clear_all()"
```

**Q: Как ввести дробную сумму?**  
Можно использовать как точку (`1500.50`), так и запятую (`1500,50`) — оба варианта поддерживаются.

**Q: Поддерживаются ли валюты кроме рублей?**  
Приложение хранит числа без символа валюты. Вы можете использовать любую валюту,
просто помните о единообразии при вводе.
